# GLTF Walk

Displays an animated GLB character walking across a simple scene.

## Run

```sh
doof run game/samples/gltf-walk
```

Press Escape or close the window to stop the sample.

## Acknowledgements

The character model and texture are from Kenney's [Animated Characters](https://kenney.nl/assets/animated-characters) asset pack, released under CC0 1.0 Universal.
